# pickled-schema

> **Status: planned, not yet implemented.**

This directory is a placeholder for the **schema** member of the pickled-\*
family. See [the monorepo roadmap](../../docs/roadmap.md) for the expected
timeline.

## What it will be

An **LLM-to-OpenAPI / JSON Schema / Protobuf** bridge: draft contracts from
natural-language descriptions; compensating gates check for ambiguity, coverage
of stated endpoints, and drift against running APIs.

## Why it's not built yet

**pickled-bdd** ships first to validate the architectural pattern on one
weak-oracle domain end-to-end. **pickled-schema** is the primary candidate for
v0.2.

## Until then

This package is **not installable**. Track progress by watching the repository
or filing an issue under the **`pkg:schema`** label.
